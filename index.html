gigVoice

2gig Panel Voice codes

gigVoice, Version 1.2

Altho very simple but affective, this version allowes security installers to search voice codes for Gen2 2gig panels.
Type a word in the search box, for example "home" or "front" then clicking Go!
If there's a match, the 2gig voice code equivalent will be displayed. Otherwise the program will output "No entries found".

Going forward, there will be updates to the program which will allow installers to view voice codes in alpha order.
